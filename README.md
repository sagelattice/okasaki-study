# okasaki-study

Goals:

- A tour of Purely Functional Data Structures by Chris Okasaki in idiomatic modern OCaml
- Do the same tour in Rust once I validate I understand the algorithms
- Use the Rust implementation to create an LLM harness to analyze chess positions that uses efficient branch pruning and time travel. It wont beat Stockfish but it will demo practical immutable data in an agentic system

AI usage policy:

- Write all of the code myself
- Have AI review syntax or compilation errors (I'm not an OCaml or Rust expert at the time of starting this)
- Have AI recommend test cases but implement them myself
- The LLM harness will obviously use a frontier provider API
